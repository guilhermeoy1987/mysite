# MySite - Modelagem de Dados com Django

Este repositório foca na definição da estrutura de banco de dados da aplicação utilizando o Django ORM.

## O que foi realizado:
* **Criação de Models:** Desenvolvimento de classes Python para representar as entidades do sistema.
* **Mapeamento de Atributos:** Uso de tipos de campos específicos (CharField, IntegerField, DateTimeField) para cada dado.
* **Relacionamentos:** Configuração de chaves estrangeiras para conectar diferentes tabelas do banco.
* **Migrações:** Execução de comandos para criação e atualização das tabelas no banco de dados SQLite.

## Tecnologias:
* Python
* Django (ORM)
* SQLite
* Git / GitHub (Branch: models)
