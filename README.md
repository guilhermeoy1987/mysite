
# MySite - Administração Django

Este repositório foca na configuração e customização do painel administrativo do Django para gerenciamento de dados do Back-end.

## O que foi realizado:
* **Habilitação do Painel Admin:** Configuração de superusuários e acesso à interface administrativa.
* **Registro de Modelos:** Integração das classes do banco de dados com o painel de controle.
* **Customização de Tabelas:** Ajuste na exibição de colunas, filtros e campos de busca no Admin.
* **Gestão de Dados:** Prática de operações CRUD (Criação, Consulta, Edição e Exclusão) via interface gráfica.

## Tecnologias:
* Python
* Django
* SQLite (Banco de dados padrão)
* Git / GitHub (Branch: django-admin)
