# MySite - Lógica de Views e Processamento de Dados

Este repositório foca na implementação das Views do Django para gerenciar o fluxo de informações entre o banco de dados e a interface do usuário.

## O que foi realizado:
* **Criação de Views:** Desenvolvimento da lógica para processar requisições e retornar templates.
* **Manipulação de POST:** Implementação de métodos para receber e tratar dados enviados por formulários.
* **Configuração de Rotas:** Mapeamento de URLs para organizar a navegação do site.
* **Integração Back-to-Front:** Passagem de contextos dinâmicos para renderização de dados nos templates.

## Tecnologias:
* Python
* Django
* Git / GitHub (Branch: post-view)
