# MySite - Desenvolvimento de Templates e Interface

Este repositório foca na construção da camada de Front-end da aplicação Django, utilizando o sistema de templates do framework.

## O que foi realizado:
* **Herança de Templates:** Criação de uma estrutura base para reaproveitamento de código em todas as páginas.
* **Renderização Dinâmica:** Uso de Django Template Language (DTL) para exibir informações do banco de dados no HTML.
* **Lógica de Exibição:** Implementação de loops e condicionais para organizar a apresentação dos dados.
* **Integração com Views:** Recebimento e tratamento de dicionários de contexto para preenchimento das páginas.

## Tecnologias:
* Python
* Django (Templates)
* HTML5 / CSS3
* Git / GitHub (Branch: templates)
