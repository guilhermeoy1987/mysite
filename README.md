# MySite - Configuração Inicial do Projeto Django

Este repositório documenta a etapa de setup e estruturação inicial de uma aplicação web utilizando o framework Django.

## O que foi realizado:
* **Ambiente Virtual:** Configuração de venv para isolamento das dependências.
* **Inicialização do Framework:** Instalação e criação da estrutura base do projeto Django.
* **Organização de Apps:** Criação e registro da estrutura de aplicativos internos.
* **Configurações Globais:** Ajustes iniciais de banco de dados e definições de sistema (settings).

## Tecnologias:
* Python
* Django
* Git / GitHub (Branch: project-setup)
