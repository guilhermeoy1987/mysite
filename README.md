# MySite - Refinamento de Templates e Estáticos

Este repositório foca na melhoria visual e na organização de arquivos estáticos dentro de uma aplicação Django.

## O que foi realizado:
* **Configuração de Static Files:** Implementação do diretório de arquivos estáticos para suporte a CSS e Imagens.
* **Estilização de Interface:** Aplicação de estilos customizados para melhorar o layout e a usabilidade das páginas.
* **Modularização de Código:** Uso da tag `include` para separar componentes de interface e facilitar a manutenção.
* **Navegação Dinâmica:** Melhoria nos links e na estrutura de navegação entre as diferentes rotas do sistema.

## Tecnologias:
* Python
* Django
* CSS3
* Git / GitHub (Branch: templates2)
